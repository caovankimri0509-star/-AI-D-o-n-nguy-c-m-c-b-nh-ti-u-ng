# HỆ THỐNG AI CHẨN ĐOÁN TIỂU ĐƯỜNG

---

## Giới thiệu
Hệ thống AI Chẩn Đoán Tiểu Đường là ứng dụng được xây dựng trên nền tảng Antigravity,  
giúp hỗ trợ người dùng phát hiện sớm nguy cơ mắc bệnh tiểu đường.  

Ứng dụng sử dụng mô hình Machine Learning (Random Forest) được huấn luyện từ dữ liệu y tế,  
nhằm đưa ra kết quả nhanh chóng, trực quan và dễ sử dụng.

---

## Chức năng chính
- 🔍 Chẩn đoán nguy cơ tiểu đường dựa trên dữ liệu người dùng nhập  
- 📊 Hiển thị thống kê:
  - Tổng số ca  
  - Ca nguy cơ cao  
  - Ca an toàn  
- 🧠 Phân tích AI thông minh với độ chính xác cao  
- 🗂 Lưu lịch sử chẩn đoán tự động  
- 📄 Phân tích dữ liệu/giấy tờ y tế  
- 💬 Trợ lý AI hỗ trợ người dùng  

---

## Công nghệ sử dụng
- ⚙️ Nền tảng phát triển: Antigravity  
- 🤖 AI/Machine Learning: Random Forest  
- 🐍 Ngôn ngữ AI: Python  
- 🌐 Frontend:
  - HTML  
  - CSS  
  - JavaScript (tự động từ Antigravity)  
- 🗄 Lưu trữ dữ liệu: Database tích hợp / API
